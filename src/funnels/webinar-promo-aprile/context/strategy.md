# Funnel Context — Webinar promo aprile

## Funnel Identity
- Slug: webinar-promo-aprile
- Funnel type: Webinar live → acquisizione lista / vendita post-webinar
- Primary objective:
  - #1: iscrizioni webinar (opt-in)
  - #2: vendite del corso post-webinar
  - #3: attivazione nel Golden Magnet (AI Playground) per onboarding + proof + nurturing

## ICP
- Segment:
  - Professionisti, freelance, piccoli imprenditori/operatori che hanno lavoro ripetitivo "da knowledge work"
  - Esempi: marketing, sales, ops, admin, ricerca, reporting, CRM, follow-up
- Awareness level: Problem Aware / Solution Aware (mix)
  - Hanno già provato ChatGPT o tool AI, ma non hanno un sistema stabile nel lavoro reale
- Core pain points:
  1) "Ho provato l'AI ma poi torno al mio modo vecchio, non si integra nel workflow"
  2) "Non so cosa usare e come metterlo a processo, ogni mese cambia tutto"
  3) "Perdo troppe ore su task ripetitivi, mi serve una soluzione affidabile"
- Desired outcomes:
  - Recuperare ore ogni settimana con un sistema che esegue lavoro ricorrente
  - Avere un metodo replicabile (non dipendente dal tool del mese)
  - Uscire dal caos "prompting random" e passare a un workflow operativo

## Positioning
- Market category:
  - Applied AI Systems for Operators (AI Operator / AI Systems Operator)
  - Non "impara l'AI", non "make money with AI"
- Unique mechanism:
  - AI WORKFORCE ENGINE: sistema ripetibile per progettare, costruire, deployare e mantenere un "AI Employee" su un task reale
- Main differentiator:
  - Approccio assets-first + deployment reale: deliverable, SOP, template, workflow pronti
  - System > tools: stabilità operativa anche se i tool cambiano
- Competitive alternatives:
  - Corsi generici su AI/ChatGPT e prompting
  - Tutorial YouTube, "tool list", blog post
  - Agenzie/automazioni spot senza metodo trasferibile
  - "Smanettamento" personale senza processo e senza asset
